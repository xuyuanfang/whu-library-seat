# 软件更新

  • 桌面端 软件启动的时候会自动访问GitHub Release检查更新，如果有更新，在软件的左下角会出现更新的按钮

  • 移动端(Android) 软件启动的时候会自动访问这个文件检查更新，如果有更新，在软件的左下角会出现下载更新的按钮
